MQI System

A Bootstrap-backed, Explorer-1-aligned design system for manufacturing quote operations. The docs shell follows Storybook-style documentation chrome while the components are themed into MQI.
Principles
Interaction model
  • upload first
  • presume confidence
  • show blockers before detail
  • optimize for quote speed
  • collapse low-value system granularity

Accessibility
  • Use strong contrast for primary text and controls.
  • Never rely on color alone for status meaning.
  • Keep dense workflows keyboard reachable.
  • Collapsed detail patterns must not block primary tasks.
  • Reserve muted text for secondary context only.
